## How to Use These Translations

### In Your Code:
```typescript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation('common');
  
  return (
    <div>
      <button>{t('buttons.add')}</button>
      <h1>{t('messages.loading')}</h1>
      <p>{t('time.today')}</p>
    </div>
  );
}
```

### Available Namespaces:
- `'common'` - General UI elements (buttons, messages, time, units)
- `'navigation'` - Navigation, tabs, menus, greetings
- `'recipe'` - Recipe-specific UI (library, details, categories)
- `'shopping'` - Shopping list and prep hub
- `'onboarding'` - Welcome and setup screens
